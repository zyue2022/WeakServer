# weakServer

linux系统上的并发服务器

### 使用
编译：`g++ *.cpp -o app -pthread -Wall` 

运行：`./app 端口号` ，然后浏览器打开网址`http://127.0.0.1:端口号/index.html/`

- 同步IO模拟proactor模式;
- 采用IO多路复用技术epoll的边缘触发模式；
- 主线程负责数据读写操作；
- 子线程负责对请求进行逻辑处理；
- 子线程使用一个线程池来管理；
- 采用有限状态机来解析http请求，暂时只支持GET；
- 添加了基于升序链表的定时器来关闭超时连接；
- 添加了异步日志系统模块;
